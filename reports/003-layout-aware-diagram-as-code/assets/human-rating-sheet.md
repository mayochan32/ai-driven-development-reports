- [ ] ノード、ラベル、矢印の重なり

## 読解確認

1. 決済開始の条件は何か。
2. 注文保存失敗時に何が起きるか。
3. 配送失敗と通知失敗は注文結果へ影響するか。
4. 最も誤読しやすかった箇所はどこか。

## 校正方法

3人以上が独立評価した後に、項目別の中央値、最大差、必須違反の一致率を集計する。点差が2以上の項目は、アンカー例と判定文を見直してから再評価する。
