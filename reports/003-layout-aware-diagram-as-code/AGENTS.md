# AGENTS.md

このファイルは、`003-layout-aware-diagram-as-code`テーマ内で作業するAIエージェント向けの追加ルールです。

## スコープ

- このテーマに固有のレポート、図、評価データ、スキル、エージェント、ツール、依存定義は、このディレクトリ内で完結させる。
- 実行コマンドは、このディレクトリを作業ルートとして記述する。
- 他テーマでも実際に使うことが確認されるまで、ファイルをリポジトリルートの共通領域へ移さない。

## 正本と派生物

- 考察の正本は `003-layout-aware-diagram-as-code.md` とする。
- `003-layout-aware-diagram-as-code.html` は人間向けの派生ビューとする。
- 図のソース、PNG、SVG、評価YAML、計測YAMLは `assets/` に置く。
- 図のソースを変更したら実際に再レンダリングし、HTMLへ埋め込んだ画像とソースも同期する。

## 検証

- コマンドは原則としてこのディレクトリから実行する。
- HTML変更後は `npm run verify:report` を実行する。
- JSON、YAML、JavaScriptの構文と、HTML内のタブ、画像、ソース対応を確認する。
- レンダリング成功だけで意味や視覚品質が正しいと判定しない。
